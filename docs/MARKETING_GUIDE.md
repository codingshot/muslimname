# Marketing Guide — MuslimName.me

## Marketing Folder

The `marketing/` folder contains reusable copy and strategy:

| File | Purpose |
|------|---------|
| `VALUE_PROPOSITIONS.md` | Headlines, value props, taglines, CTAs |
| `SOCIAL_SNIPPETS.md` | Twitter/X, Instagram, LinkedIn captions; hashtags; meta descriptions |
| `SEO_KEYWORDS.md` | Primary, long-tail, and geo keywords for SEO |

## How to Use

1. **Landing page copy** — Pull from VALUE_PROPOSITIONS.md for hero, feature blocks, and CTAs
2. **Social posts** — Use SOCIAL_SNIPPETS.md for quick posts; adjust for platform
3. **SEO** — Reference SEO_KEYWORDS.md when writing blogs, meta tags, and alt text
4. **New features** — Update value props and snippets when adding major features (e.g. Korean names, fiqh detection)

## Key Messages

- **Audience:** New Muslims (reverts), converts, anyone exploring Islamic names
- **Tone:** Warm, inclusive, educational, respectful of all Abrahamic traditions
- **Differentiators:** 1700+ names, Quran/Bible/Torah links, 40+ country legal guides, fiqh detection, multilingual (Korean, Chinese, Thai, Vietnamese, etc.)
- **CTA:** Start your name journey, Discover your Islamic equivalent, Find names like [Name]
